# Spec: Language Life Art

## 1. Goal

言語を中心に据えた無限生成の Web アートを公開し、閲覧者が「生命の同時性」「渾然一体」「永遠と愛」の感覚を体験できること。

## 2. Scope

- 静的サイト（`index.html`, `style.css`, `script.js`）
- ブラウザ上で無限に変容する抽象表現
- 生成文は生命/自然/時間/愛を主題にした言語構成
- ユーザーは「見る」または「カーソル移動」のみで体験できる
- GitHub Pages で公開可能な構成

## 3. Out of Scope

- サーバーサイド処理
- 永続ストレージ（DB）
- ユーザー認証

## 4. Acceptance Criteria

1. ページ読み込み後、Canvas上で抽象ビジュアルと文片が自動生成され続ける。
2. 文片は時間とともに更新され、停止しない（無限体験）。
3. AI API を使用せず、ブラウザ内のみで生成が完結する。
4. カーソル移動に応じて、文片の発生位置や色/接続が変化する。
5. 入力フォームやクリック操作なしでも体験が成立する。
6. モバイル幅でも主要表示が崩れず、閲覧体験が成立する。
7. `main` pushでGitHub Pagesデプロイが走るワークフローが存在する。

## 5. Definition of Done

- 受入条件 1-7 がローカル確認で成立
- README に実行方法と操作方法を明記
- 仕様・意思決定・進捗が `docs` と `progress.md` に記録されている
