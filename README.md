# Language Life Art

言語を軸に、終わりのない抽象体験を生む Web アートです。  
生成される文片は、生命・自然・時間・愛をテーマに変容し続けます。

コンセプト:

> 生命活動は自然現象で、同時間を過ごす生命は渾然一体なので、永遠も愛も存在する。

## Run

静的サイトなので、任意のHTTPサーバーで開けます。

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開きます。

## Experience

- 基本は「見るだけ」で体験が進行
- 手の動きに応じて、文片密度・色相・接続線の振る舞いが変化
- 日本語 / English / Espanol / Francais を横断して文片が再構成される
- 関連線は `直線 / グラフ線 / 波線` を行き来し、関係性を可視化する
- AI API は不使用（完全クライアントサイド生成）

## Deploy

`main` ブランチへ push すると、GitHub Pages ワークフローで自動公開されます。
